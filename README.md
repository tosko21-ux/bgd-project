# BGD — Bike Gear Decoder

Tool-first web for cyclists. Quick answers, interactive comparisons, no fluff.

## Status

🚧 Early development (April 2026)

## Stack

- HTML + CSS + vanilla JS (Phase 1)
- Cloudflare Pages (planned)

## Local development

1. Open `src/pages/index.html` with VS Code Live Server extension
2. Edits auto-reload in browser

## Structure

- `src/pages/` — HTML pages, one per URL
- `src/styles/` — CSS files
- `src/scripts/` — JavaScript
- `src/data/` — JSON data (groupsets etc.)
- `public/` — static assets (images)
